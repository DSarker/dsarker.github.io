Projects

RandoChat


Android text based chat app that lets you anonymously connect with random people for a one on one conversation. No sign up required. Simply enter a username and start chatting. Swipe left to change rooms. It's easy and fun to use.

Lister

Android app that lets users keep track of all their tasks. It has a simple, clean design and is easy to use. The user can create, update, and delete tasks. The user can also see when a task needs to be completed, mark a task as complete, or delete all completed tasks with ease. This is for people that need to get things done.

Vice City

Android application that displays news articles using the Vice API. The user sees articles based on category which can be switched with tabs. The user can also search for articles based on title and share articles with Facebook using the Facebook API.

MyPlaces

Android app that includes a SQLite database of locations that can be searched for by name, type, and borough. The user can save a location to their favorites. The favorites list is easily accessible by clicking the star button. The user is able to write reviews which are then saved in the database. The average rating and number of reviews is updated accordingly.
